Add shortcodes for Doodle (Hand-drawn Wordpress Theme).

== Description ==

The plugin will add shortcodes for:

* Headings with icons
* Features
* Buttons
* Countdowns
* Portfolio sliders
* Testimonials

== Installation ==

1. Upload the `doodle-shortcodes` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
