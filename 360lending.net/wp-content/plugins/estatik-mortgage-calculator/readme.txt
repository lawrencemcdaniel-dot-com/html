It is great-looking and informative, and absolutely free! Check out http://estatik.net/ for more information.Visit [demo](http://demo.estatik.net/) of Estatik calculator on one of default WordPress themes or [demo](http://trendy.estatik.net/) on Estatik custom theme.Main features:* Estatik mortgage calculator looks good on any device from both sides: admin and front.* You can use it as simple calculator with Purchase price, Down payment, Interest Rate or enable advanced options and switch on extra Home Insurance, Property taxes and PMI.* Adapt calculator color according to your theme styles. Choose any color you need via plugin Settings.* The calculator displays results in Popup window with graph or simple text and image. You can select one of options most appealing for you.* Add calculator to your sidebars as widget or use shortcode to add it to any page.* To make it more attractive we used jQuery and Google Charts in its code.* If used with [Estatik plugin](https://wordpress.org/plugins/estatik/), Purchase Price field in Calculator is synchronized with Listing price in Estatik if used together on the same page.Translated into:* Russian* Spanish* German* Italian* DutchVisit official [Estatik Mortgage Calculator page](https://estatik.net/product/estatik-mortgage-calculator/) to get full information.== Installation ==1. Upload files of Estatik Calculator plugin to the `/wp-content/plugins/` 2. Activate the plugin through the 'Plugins' menu in WordPress3. Go to widgets and add Estatik Mortgage Calculator Widget to selected Sidebar OR use shortcode [es_mortgage_calculator layout="horizontal/vertical"] to add calculator to the page4. Go to Settings and find Estatik Mortgage Calculator there to configure it == Frequently Asked Questions === Does Estatik Calculator work with any Wordpress theme? =Yes, it does.
